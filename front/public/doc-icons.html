Icons

Usage

1. Configure the list of the svg icons in the config file ce-icons.config.js
export const icons = {
  user: `
    <svg viewBox="0 0 24 24">
      <path fill="currentColor" d="M12 12q-1.65 0-2.825-1.175T8 8q0-1.65 1.175-2.825T12 4q1.65 0 2.825 1.175T16 8q0 1.65-1.175 2.825T12 12m-8 8v-2.8q0-.85.438-1.562T5.6 14.55q1.55-.775 3.15-1.162T12 13q1.65 0 3.25.388t3.15 1.162q.725.375 1.163 1.088T20 17.2V20z"/>
    </svg>
  `,
  // ...
};

TIP: Find many icones on icones.js.org

2. Import the icon component in your javascript file
import "./components/ce-icon.js";
3. Use the icon component in your HTML file
<ce-icon name="user"></ce-icon>
Result:

Colors

You can change the color of the icon by setting the CSS color property.

Code:
<ce-icon name="user"></ce-icon>
<ce-icon name="user" style="color: green;"></ce-icon>
<ce-icon name="user" style="color: red;"></ce-icon>
Result:

Size

You can change the size of the icon by setting the CSS font-size property.

Code:
<ce-icon name="user"></ce-icon>
<ce-icon name="user" style="font-size: 1.5rem;"></ce-icon>
<ce-icon name="user" style="font-size: 3rem;"></ce-icon>
Result:

Scale

You can adjust the size of the icon by setting the scale attribute.

Code:
<ce-icon name="user" scale="1.5"></ce-icon> My account
Result:
My account

Block

You can remove the inline behaviour of the icon by adding the block attribute.

Code:
<ce-icon name="user" scale="1.5" block></ce-icon> My account
Result:
My account

Rotate

You can rotate the icon by setting the rotate attribute.

Code:
<ce-icon name="user"></ce-icon>
<ce-icon name="user" rotate="45"></ce-icon>
<ce-icon name="user" rotate="90"></ce-icon>
<ce-icon name="user" rotate="135"></ce-icon>
<ce-icon name="user" rotate="180"></ce-icon>
